# Log Monitoring & Analysis

## Objective
Analyze system logs to detect suspicious activities and security incidents.

## Tools Used
- Linux authentication logs
- Windows Event Viewer (optional)
- Splunk Free (optional)

## Workflow
1. Understand log types
2. Analyze authentication logs
3. Detect failed login attempts
4. Identify anomalies
5. Correlate events
6. Learn SIEM basics
7. Create alert rules
8. Document findings
